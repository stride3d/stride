# Stride.GameStudio.Avalonia.Desktop

This project is for the final executable for the Game Studio that uses Avalonia as a UI library.

## Dependencies

* It should reference `Stride.GameStudio.Avalonia` as well as `Avalonia.Desktop`.
* It is free to have any other kind of dependencies as long as they are cross-platform with regard to a Desktop target.

## Implementations

Beside the main `Program.cs`, it is not expected to have other code file.
